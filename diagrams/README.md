# Diagram Export Files

This folder contains class diagram source code files that can be rendered on various diagram platforms.

## Files

| File | Format | Where to Use |
|------|--------|--------------|
| `class_diagram.puml` | PlantUML | [PlantUML Online](https://www.plantuml.com/plantuml/uml/), VS Code PlantUML Extension, IntelliJ |
| `class_diagram.mmd` | Mermaid | [Mermaid Live Editor](https://mermaid.live/), GitHub Markdown, VS Code |
| `class_diagram.drawio` | Draw.io | [Draw.io](https://app.diagrams.net/), VS Code Draw.io Extension |

## How to Use

### PlantUML (Recommended for detailed diagrams)

1. Go to [PlantUML Online Server](https://www.plantuml.com/plantuml/uml/)
2. Paste the contents of `class_diagram.puml`
3. Click "Submit" to render
4. Download as PNG, SVG, or PDF

**Or use VS Code:**
1. Install "PlantUML" extension
2. Open `class_diagram.puml`
3. Press `Alt + D` to preview

### Mermaid (Good for quick previews)

1. Go to [Mermaid Live Editor](https://mermaid.live/)
2. Paste the contents of `class_diagram.mmd`
3. See live preview and export as PNG/SVG

**Or in GitHub:**
- Mermaid diagrams render automatically in GitHub Markdown files

### Draw.io (For manual editing)

1. Go to [Draw.io](https://app.diagrams.net/)
2. File > Import > Select `class_diagram.drawio`
3. Edit visually and export to any format
